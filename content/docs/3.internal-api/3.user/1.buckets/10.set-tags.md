| Endpoint | Method | Admin Required |
| -------- | ------ | -------------- |
| `/api/v1/buckets/:bucketName/tags` | :badge{label="PUT" color="amber"} | :badge{label="No" color="secondary"} |

Sets the tags for a bucket you own, **overwriting any existing tags entirely** - this does not merge with the current tag-set. Matches the S3-compatible API's `PutBucketTagging` semantics.

## Path Parameters

::field-group
    ::field{name="bucketName" type="string" required}
    The name of the bucket.
    ::
::

## Request Body

::field-group
    ::field{name="tags" type="object" required}
    A flat key-value map of tags. Send an empty object to clear all tags (equivalent to [Delete Bucket Tags](/docs/internal-api/user/buckets/delete-tags)).
    ::
::

### Example

```json
{
    "tags": {
        "env": "prod",
        "team": "storage"
    }
}
```

## Response

Returns the saved tags:

```json
{
    "tags": {
        "env": "prod",
        "team": "storage"
    }
}
```

## Errors

| Status | Reason |
| ------ | ------ |
| `400` | Missing bucket name |
| `404` | Bucket not found (or not owned by you) |
