| Endpoint | Method | Admin Required |
| -------- | ------ | -------------- |
| `/api/v1/buckets/:bucketName/tags` | :badge{label="DELETE" color="error"} | :badge{label="No" color="secondary"} |

Removes all tags from a bucket you own.

## Path Parameters

::field-group
    ::field{name="bucketName" type="string" required}
    The name of the bucket.
    ::
::

## Response

Returns `204 No Content` on success.

## Errors

| Status | Reason |
| ------ | ------ |
| `400` | Missing bucket name |
| `404` | Bucket not found (or not owned by you) |
