| Endpoint | Method | Admin Required |
| -------- | ------ | -------------- |
| `/api/v1/buckets` | :badge{label="GET" color="success"} | :badge{label="No" color="secondary"} |

Returns a paginated list of buckets owned by the authenticated user.

## Query Parameters

::field-group
    ::field{name="page" type="integer"}
    Page number. Default: `1`
    ::

    ::field{name="per" type="integer"}
    Items per page. Default: `10`
    ::
::

## Response

```json
{
    "metadata": {
        "page": 1,
        "per": 10,
        "total": 2
    },
    "items": [
        {
            "id": "B1C2D3E4-F5G6-7890-HIJK-LM1234567890",
            "name": "my-assets",
            "creationDate": "2025-01-15T10:30:00Z",
            "versioningStatus": "Enabled"
        },
        {
            "id": "C2D3E4F5-G6H7-8901-IJKL-MN2345678901",
            "name": "backups",
            "creationDate": "2025-01-10T08:15:00Z",
            "versioningStatus": "Disabled"
        }
    ]
}
```

## Response Fields

::field-group
    ::field{name="id" type="string"}
    Unique bucket identifier (UUID).
    ::

    ::field{name="name" type="string"}
    Bucket name.
    ::

    ::field{name="creationDate" type="string"}
    ISO 8601 timestamp of when the bucket was created.
    ::

    ::field{name="versioningStatus" type="string"}
    Versioning status: `Disabled`, `Enabled`, or `Suspended`.
    ::
::
