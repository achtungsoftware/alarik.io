| Endpoint | Method | Admin Required |
| -------- | ------ | -------------- |
| `/api/v1/buckets/:bucketName` | :badge{label="DELETE" color="error"} | :badge{label="No" color="secondary"} |

Permanently deletes a bucket and all its contents. You can only delete buckets you own.

::warning
This action is irreversible. All objects in the bucket will be permanently deleted.
::

## Path Parameters

::field-group
    ::field{name="bucketName" type="string" required}
    The name of the bucket to delete.
    ::
::

## Response

Returns `204 No Content` on success.

## Errors

| Status | Reason |
| ------ | ------ |
| `400` | Missing bucket name |
| `404` | Bucket not found (or not owned by you) |
