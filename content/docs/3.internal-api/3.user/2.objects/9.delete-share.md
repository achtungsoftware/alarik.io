| Endpoint | Method | Admin Required |
| -------- | ------ | -------------- |
| `/api/v1/objects/share/:sharedLinkId` | :badge{label="DELETE" color="error"} | :badge{label="No" color="secondary"} |

Revokes a shared link immediately, before it would otherwise expire. Anyone using the link afterwards gets a `404`.

## Path Parameters

::field-group
    ::field{name="sharedLinkId" type="string" required}
    The shared link's `id`, as returned by [Share Object](/docs/internal-api/user/objects/share) or [List Shared Links](/docs/internal-api/user/objects/list-shares).
    ::
::

## Response

Returns `204 No Content` on success.

## Errors

| Status | Reason |
| ------ | ------ |
| `400` | `sharedLinkId` is not a valid UUID |
| `404` | Shared link not found (or not owned by you) |
