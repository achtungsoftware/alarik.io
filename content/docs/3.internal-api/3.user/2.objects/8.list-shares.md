| Endpoint | Method | Admin Required |
| -------- | ------ | -------------- |
| `/api/v1/objects/share` | :badge{label="GET" color="success"} | :badge{label="No" color="secondary"} |

Returns a paginated list of shared links you've created, across all of your buckets.

## Query Parameters

::field-group
    ::field{name="page" type="integer"}
    Page number. Default: `1`
    ::

    ::field{name="per" type="integer"}
    Items per page. Default: `10`
    ::
::

## Response

```json
{
    "metadata": {
        "page": 1,
        "per": 10,
        "total": 1
    },
    "items": [
        {
            "id": "3fa85f64-5717-4562-b3fc-2c963f66afa6",
            "bucketName": "my-bucket",
            "key": "report.pdf",
            "url": "https://api.example.com/api/v1/shared/3fa85f64-5717-4562-b3fc-2c963f66afa6",
            "expiresAt": "2025-01-15T13:00:00Z",
            "createdAt": "2025-01-15T12:00:00Z"
        }
    ]
}
```

## Response Fields

::field-group
    ::field{name="id" type="string"}
    Unique identifier of the shared link. Also the token used in `url`.
    ::

    ::field{name="bucketName" type="string"}
    Name of the bucket the shared object belongs to.
    ::

    ::field{name="key" type="string"}
    Key of the shared object.
    ::

    ::field{name="url" type="string"}
    The public, unauthenticated URL. See [Share Object](/docs/internal-api/user/objects/share).
    ::

    ::field{name="expiresAt" type="string"}
    ISO 8601 timestamp of when the link stops working.
    ::

    ::field{name="createdAt" type="string"}
    ISO 8601 timestamp of when the link was created.
    ::
::
