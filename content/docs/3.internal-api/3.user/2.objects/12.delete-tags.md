| Endpoint | Method | Admin Required |
| -------- | ------ | -------------- |
| `/api/v1/objects/tags` | :badge{label="DELETE" color="error"} | :badge{label="No" color="secondary"} |

Removes all tags from the *current* version of an object in a bucket you own.

## Query Parameters

::field-group
    ::field{name="bucket" type="string" required}
    Name of the bucket.
    ::

    ::field{name="key" type="string" required}
    Object key.
    ::
::

## Response

Returns `204 No Content` on success.

## Errors

| Status | Reason |
| ------ | ------ |
| `400` | Missing `bucket` or `key` parameter |
| `404` | Bucket not found (or not owned by you), or object not found |
