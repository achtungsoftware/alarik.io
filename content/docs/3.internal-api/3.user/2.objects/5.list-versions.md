| Endpoint | Method | Admin Required |
| -------- | ------ | -------------- |
| `/api/v1/objects/versions` | :badge{label="GET" color="success"} | :badge{label="No" color="secondary"} |

Returns all versions of a specific object, including delete markers. Only works on buckets where versioning has been enabled at some point.

## Query Parameters

::field-group
    ::field{name="bucket" type="string" required}
    Name of the bucket.
    ::

    ::field{name="key" type="string" required}
    Object key to list versions for.
    ::
::

## Response Fields

::field-group
    ::field{name="versionId" type="string"}
    Unique identifier for this version.
    ::

    ::field{name="isLatest" type="boolean"}
    Whether this is the current (latest) version.
    ::

    ::field{name="isDeleteMarker" type="boolean"}
    Whether this version is a delete marker (indicates the object was deleted).
    ::
::

## Errors

| Status | Reason |
| ------ | ------ |
| `400` | Missing `bucket` or `key` parameter |
| `404` | Bucket not found (or not owned by you) |
