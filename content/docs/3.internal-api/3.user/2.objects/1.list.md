| Endpoint | Method | Admin Required |
| -------- | ------ | -------------- |
| `/api/v1/objects` | :badge{label="GET" color="success"} | :badge{label="No" color="secondary"} |

Returns a paginated list of objects in a bucket you own. Supports prefix filtering for navigating folder structures.

## Query Parameters

::field-group
    ::field{name="bucket" type="string" required}
    Name of the bucket to list objects from.
    ::

    ::field{name="prefix" type="string"}
    Filter objects by key prefix. Use for folder navigation (e.g., `images/`).
    ::

    ::field{name="delimiter" type="string"}
    Character to group keys. Default: `/`. Used to simulate folder hierarchy.
    ::

    ::field{name="page" type="integer"}
    Page number. Default: `1`
    ::

    ::field{name="per" type="integer"}
    Items per page. Default: `100`
    ::
::

## Response

```json
{
    "metadata": {
        "page": 1,
        "per": 100,
        "total": 5
    },
    "items": [
        {
            "key": "documents/",
            "size": 0,
            "contentType": "application/x-directory",
            "etag": "",
            "lastModified": "2025-01-15T10:30:00Z",
            "isFolder": true
        },
        {
            "key": "image.png",
            "size": 245678,
            "contentType": "image/png",
            "etag": "d41d8cd98f00b204e9800998ecf8427e",
            "lastModified": "2025-01-14T15:22:00Z",
            "isFolder": false,
            "versionId": "abc123def456",
            "isLatest": true,
            "isDeleteMarker": false
        }
    ]
}
```

## Response Fields

::field-group
    ::field{name="key" type="string"}
    Object key (path). Folders end with `/`.
    ::

    ::field{name="size" type="integer"}
    Object size in bytes. `0` for folders.
    ::

    ::field{name="contentType" type="string"}
    MIME type. `application/x-directory` for folders.
    ::

    ::field{name="etag" type="string"}
    MD5 hash of the object content.
    ::

    ::field{name="lastModified" type="string"}
    ISO 8601 timestamp of last modification.
    ::

    ::field{name="isFolder" type="boolean"}
    Whether this entry represents a folder (common prefix).
    ::

    ::field{name="versionId" type="string"}
    Version identifier (if versioning is enabled).
    ::

    ::field{name="isLatest" type="boolean"}
    Whether this is the latest version.
    ::

    ::field{name="isDeleteMarker" type="boolean"}
    Whether this version is a delete marker.
    ::
::

## Errors

| Status | Reason |
| ------ | ------ |
| `400` | Missing `bucket` query parameter |
| `404` | Bucket not found (or not owned by you) |
