User endpoints work on resources owned by the authenticated user. No admin privileges required.

## Available Endpoints

::card-group
  ::card{title="Buckets" icon="i-lucide-cylinder" to="/docs/internal-api/user/buckets/list"}
  Create, list, delete buckets and manage versioning.
  ::

  ::card{title="Objects" icon="i-lucide-file" to="/docs/internal-api/user/objects/list"}
  Upload, download, list, and delete objects.
  ::
::
