| Endpoint | Method | Admin Required |
| -------- | ------ | -------------- |
| `/api/v1/admin/oidcProviders` | :badge{label="GET" color="success"} | :badge{label="Yes" color="warning"} |

Returns a paginated list of all configured OIDC providers, including disabled ones. The client secret is never included in the response - see [Create OIDC Provider](/docs/internal-api/admin/oidc-providers/create).

## Query Parameters

::field-group
    ::field{name="page" type="integer"}
    Page number. Default: `1`
    ::

    ::field{name="per" type="integer"}
    Items per page. Default: `10`
    ::
::

## Response

```json
{
    "metadata": {
        "page": 1,
        "per": 10,
        "total": 1
    },
    "items": [
        {
            "id": "63717CB9-DE10-44D2-AC69-774F57F6C148",
            "name": "Google",
            "issuerURL": "https://accounts.google.com",
            "clientId": "your-client-id",
            "enabled": true
        }
    ]
}
```

## Response Fields

::field-group
    ::field{name="id" type="string"}
    Unique provider identifier (UUID).
    ::

    ::field{name="name" type="string"}
    Display label shown on the login page.
    ::

    ::field{name="issuerURL" type="string"}
    The provider's OIDC issuer URL.
    ::

    ::field{name="clientId" type="string"}
    The OAuth client ID registered with the provider.
    ::

    ::field{name="enabled" type="boolean"}
    Whether the provider is shown as a sign-in option.
    ::
::
