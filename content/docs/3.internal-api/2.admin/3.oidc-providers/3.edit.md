| Endpoint | Method | Admin Required |
| -------- | ------ | -------------- |
| `/api/v1/admin/oidcProviders` | :badge{label="PUT" color="amber"} | :badge{label="Yes" color="warning"} |

Updates an existing OIDC provider's configuration.

::note
`clientSecret` is optional on edit - omit it (or send an empty string) to keep the existing secret unchanged. Send a value to rotate it.
::

## Request Body

::field-group
    ::field{name="id" type="string (UUID)" required}
    The unique identifier of the provider to update.
    ::

    ::field{name="name" type="string" required}
    New display label.
    ::

    ::field{name="issuerURL" type="string" required}
    New issuer URL.
    ::

    ::field{name="clientId" type="string" required}
    New OAuth client ID.
    ::

    ::field{name="clientSecret" type="string"}
    New OAuth client secret. Omit to keep the existing one.
    ::

    ::field{name="enabled" type="boolean" required}
    Whether the provider should be shown as a sign-in option.
    ::
::

### Example

```json
{
    "id": "63717CB9-DE10-44D2-AC69-774F57F6C148",
    "name": "Google Workspace",
    "issuerURL": "https://accounts.google.com",
    "clientId": "your-client-id",
    "enabled": false
}
```

## Response

Returns the updated provider (without the client secret):

```json
{
    "id": "63717CB9-DE10-44D2-AC69-774F57F6C148",
    "name": "Google Workspace",
    "issuerURL": "https://accounts.google.com",
    "clientId": "your-client-id",
    "enabled": false
}
```

## Errors

| Status | Reason |
| ------ | ------ |
| `400` | Validation failed (invalid UUID, empty name, issuer URL, or client ID) |
| `404` | Provider not found |
