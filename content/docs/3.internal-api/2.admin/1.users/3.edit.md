| Endpoint | Method | Admin Required |
| -------- | ------ | -------------- |
| `/api/v1/admin/users` | :badge{label="PUT" color="amber"} | :badge{label="Yes" color="warning"} |

Updates an existing user's profile information.

::note
This endpoint does not update the user's password. Password changes should be handled separately.
::

## Request Body

::field-group
    ::field{name="id" type="string (UUID)" required}
    The unique identifier of the user to update.
    ::

    ::field{name="name" type="string" required}
    New display name for the user.
    ::

    ::field{name="username" type="string" required}
    New username. Must be unique across all users.
    ::

    ::field{name="isAdmin" type="boolean" required}
    Whether the user should have admin privileges.
    ::
::

### Example

```json
{
    "id": "A1B2C3D4-E5F6-7890-ABCD-EF1234567890",
    "name": "John Smith",
    "username": "johnsmith",
    "isAdmin": true
}
```

## Response

Returns the updated user:

```json
{
    "id": "A1B2C3D4-E5F6-7890-ABCD-EF1234567890",
    "name": "John Smith",
    "username": "johnsmith",
    "isAdmin": true
}
```

## Errors

| Status | Reason |
| ------ | ------ |
| `400` | Validation failed (invalid UUID, empty name or username) |
| `409` | Username already exists (taken by another user) |
