| Endpoint | Method | Admin Required |
| -------- | ------ | -------------- |
| `/api/v1/admin/users` | :badge{label="GET" color="success"} | :badge{label="Yes" color="warning"} |

Returns a paginated list of all users in the system.

## Query Parameters

::field-group
    ::field{name="page" type="integer"}
    Page number. Default: `1`
    ::

    ::field{name="per" type="integer"}
    Items per page. Default: `10`
    ::
::

## Response

```json
{
    "metadata": {
        "page": 1,
        "per": 10,
        "total": 2
    },
    "items": [
        {
            "id": "63717CB9-DE10-44D2-AC69-774F57F6C148",
            "name": "Admin User",
            "username": "alarik",
            "isAdmin": true
        },
        {
            "id": "A1B2C3D4-E5F6-7890-ABCD-EF1234567890",
            "name": "Regular User",
            "username": "john",
            "isAdmin": false
        }
    ]
}
```

## Response Fields

::field-group
    ::field{name="id" type="string"}
    Unique user identifier (UUID).
    ::

    ::field{name="name" type="string"}
    Display name of the user.
    ::

    ::field{name="username" type="string"}
    Unique username for authentication.
    ::

    ::field{name="isAdmin" type="boolean"}
    Whether the user has admin privileges.
    ::
::
