| Endpoint | Method | Admin Required |
| -------- | ------ | -------------- |
| `/api/v1/admin/users` | :badge{label="POST" color="info"} | :badge{label="Yes" color="warning"} |

Creates a new user account.

## Request Body

::field-group
    ::field{name="name" type="string" required}
    Display name of the user.
    ::

    ::field{name="username" type="string" required}
    Unique username for authentication. Must be unique across all users.
    ::

    ::field{name="password" type="string" required}
    Password for the user account.
    ::

    ::field{name="isAdmin" type="boolean" required}
    Whether the user should have admin privileges.
    ::
::

### Example

```json
{
    "name": "John Doe",
    "username": "john",
    "password": "securePassword123",
    "isAdmin": false
}
```

## Response

Returns the created user (without password):

```json
{
    "id": "A1B2C3D4-E5F6-7890-ABCD-EF1234567890",
    "name": "John Doe",
    "username": "john",
    "isAdmin": false
}
```

## Errors

| Status | Reason |
| ------ | ------ |
| `400` | Validation failed (empty name, username, or password) |
| `409` | Username already exists |
