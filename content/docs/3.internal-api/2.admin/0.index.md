Admin endpoints require the authenticated user to have admin privileges. They provide system-wide access to manage all users and buckets.

## Available Endpoints

::card-group
  ::card{title="Users" icon="i-lucide-users" to="/docs/internal-api/admin/users/list"}
  Create, list, edit, and delete user accounts.
  ::

  ::card{title="Buckets" icon="i-lucide-cylinder" to="/docs/internal-api/admin/buckets/list"}
  List and delete any bucket in the system.
  ::

  ::card{title="Storage Stats" icon="i-lucide-hard-drive" to="/docs/internal-api/admin/storage-stats"}
  View disk usage and system metrics.
  ::
::
