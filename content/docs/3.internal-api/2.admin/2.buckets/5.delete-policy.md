| Endpoint | Method | Admin Required |
| -------- | ------ | -------------- |
| `/api/v1/admin/buckets/:bucketName/policy` | :badge{label="DELETE" color="error"} | :badge{label="Yes" color="warning"} |

Removes the policy from any bucket, regardless of owner. Any access it previously granted to unauthenticated requests stops immediately.

## Path Parameters

::field-group
    ::field{name="bucketName" type="string" required}
    The name of the bucket.
    ::
::

## Response

Returns `204 No Content` on success.

## Errors

| Status | Reason |
| ------ | ------ |
| `400` | Missing bucket name |
| `404` | Bucket not found |
