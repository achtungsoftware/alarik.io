| Endpoint | Method | Admin Required |
| -------- | ------ | -------------- |
| `/api/v1/admin/buckets` | :badge{label="GET" color="success"} | :badge{label="Yes" color="warning"} |

Returns a paginated list of all buckets in the system, including the owner information.

::note
Unlike the [user endpoint](/docs/internal-api/user/buckets/list), this returns buckets from all users.
::

## Query Parameters

::field-group
    ::field{name="page" type="integer"}
    Page number. Default: `1`
    ::

    ::field{name="per" type="integer"}
    Items per page. Default: `10`
    ::
::

## Response Fields

::field-group
    ::field{name="id" type="string"}
    Unique bucket identifier (UUID).
    ::

    ::field{name="name" type="string"}
    Bucket name.
    ::

    ::field{name="creationDate" type="string"}
    ISO 8601 timestamp of when the bucket was created.
    ::

    ::field{name="versioningStatus" type="string"}
    Versioning status: `Disabled`, `Enabled`, or `Suspended`.
    ::

    ::field{name="user" type="object"}
    Owner information including `id`, `name`, and `username`.
    ::
::
