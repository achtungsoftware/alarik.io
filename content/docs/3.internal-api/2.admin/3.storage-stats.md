| Endpoint | Method | Admin Required |
| -------- | ------ | -------------- |
| `/api/v1/admin/storageStats` | :badge{label="GET" color="success"} | :badge{label="Yes" color="warning"} |

Returns storage statistics including disk usage, bucket count, and user count.

## Response

```json
{
    "totalBytes": 500107862016,
    "availableBytes": 234567890123,
    "usedBytes": 265539971893,
    "alarikUsedBytes": 1073741824,
    "bucketCount": 15,
    "userCount": 3
}
```

## Response Fields

::field-group
    ::field{name="totalBytes" type="integer"}
    Total disk capacity in bytes.
    ::

    ::field{name="availableBytes" type="integer"}
    Available disk space in bytes.
    ::

    ::field{name="usedBytes" type="integer"}
    Total disk space used (system-wide) in bytes.
    ::

    ::field{name="alarikUsedBytes" type="integer"}
    Disk space used specifically by Alarik storage in bytes.
    ::

    ::field{name="bucketCount" type="integer"}
    Total number of buckets in the system.
    ::

    ::field{name="userCount" type="integer"}
    Total number of registered users.
    ::
::

## Example Usage

```bash
curl -X GET "{API_BASE_URL}/api/v1/admin/storageStats" \
  -H "X-Access-Key: YOUR_ACCESS_KEY" \
  -H "X-Secret-Key: YOUR_SECRET_KEY"
```
